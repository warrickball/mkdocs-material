OnyxClient

__init__(username=None, env_password=False, directory=None)

Initialise the client.

PARAMETER DESCRIPTION
username

User to act as. If not provided, acts as the default user in the config.

TYPE: str DEFAULT: None

env_password

If set to True, gets the user's password from the ONYX_<username>_PASSWORD environment variable.

TYPE: bool DEFAULT: False

directory

Path to config directory. If not provided, uses directory stored in the ONYX_CLIENT_CONFIG environment variable.

TYPE: str DEFAULT: None

Get a record from the database.

login()

Log in as a particular user, get a new token and store the token in the client.

If no user is provided, the default_user in the config is used.

logout()

Log out the user in this client.

logoutall()

Log out the user in all clients.

query(project, query=None, include=None, exclude=None, scope=None)

site_list_users()

Get the current users within the site of the requesting user.

site_list_waiting()

List users waiting for site approval.

Update a record in the database.

OnyxConfig

Class for managing the config directory (and files within) that are used by OnyxClient.

__init__(directory=None)

Initialise the config.

PARAMETER DESCRIPTION
directory

Path to config directory. If not provided, uses directory stored in the ONYX_CLIENT_CONFIG environment variable.

TYPE: str DEFAULT: None

add_user(username)

Add a new user to the config.

PARAMETER DESCRIPTION
username

Name of the user being added.

TYPE: str

get_default_user()

Get the default user in the config.

list_users()

Get a list of the users in the config.

set_default_user(username)

Set the default user in the config.

PARAMETER DESCRIPTION
username

Name of the user being set as default.

TYPE: str

write_token(username, token, expiry)

Update the tokens file for username.

PARAMETER DESCRIPTION
username

User within config who is having tokens updated.

TYPE: str

token

The token being written to their tokens file.

TYPE: str

expiry

Expiry of the token.

TYPE: str
